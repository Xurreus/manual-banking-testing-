# Test Plan Online Banking App

## 1. Objective
Ensure that the banking app works correctly, is secure, and all financial transactions function properly.

## 2. In-Scope
- Login & authentication
- Balance checking
- Transfers
- Transaction history
- Settings

## 3. Out-of-Scope
- Backend API load tests
- Performance stress testing

## 4. Testing Types
- Smoke Testing
- Functional Testing
- Boundary Testing (e.g., transfer amount limits)
- Regression Testing
- Security Validation (manual only)

## 5. Environment
- Web: Chrome latest version
- Mobile: Android (Pixel 6 virtual device)

## 6. Deliverables
- Test Cases
- Bug Reports
- Test Summary
